# Config.py

## Класс
- Config()

## Функции
- init()
    - Указывает каталоги и получает информацию об пользовательском каталоге
- validate_config()
    - Проверяет наличие конфигурационного файла в каталоге пользователя
- create_config()
    - Копирует конфигурационный файл с каталога программы в пользовательский каталог
- contains_values()
    - Проверяет параметры settings, catalogs.
    - Если их нету то выход из программы.
- validate_settings()
    - Проверяет настройки
    - Достает параметры с yaml файла
    - Если нету необходимых параметров то выход из программы. 
- validate_directories()
    - Проверяет directories
    - Достает параметры с yaml файла
    - Если нету необходимых параметров то выход из программы.
- validate_catalogs()
    - Проверяет catalogs
    - Достает параметры с yaml файла
    - Если нету необходимых параметров то выход из программы.
- main()
    - Запускает парсинг и функции validate_catalogs, validate_directories, validate_settings, contains_values
    - Так же ставит язык на английский при отсуствии параметра languages в settings
- run()
    - Запускает validate_config, и main.
    - Возвращает catalogs, settings, language
