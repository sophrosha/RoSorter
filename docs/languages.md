# Languages.py

## Описание
Модуль `languages.py` содержит словарь `LANGUAGE` с поддержкой двух языков: русского (ru-RU) и английского (en-US). Он предоставляет локализованные сообщения для различных событий и ошибок в приложении RoSorter.

## Структура
- `LANGUAGE`: Словарь с ключами для языков.
  - `'ru-RU'`: Сообщения на русском языке.
  - `'en-US'`: Сообщения на английском языке.

## Ключи сообщений
- `mission_option`: Сообщение об отсутствующей опции.
- `found_ignore`: Сообщение о найденном списке игнорируемых файлов.
- `delete_ignore`: Сообщение об удалении файла из списка.
- `deleted_ignore`: Подтверждение удаления файла.
- `searched_some_extensions`: Предупреждение о нескольких расширениях с *.
- `config_not_found`: Сообщение о ненайденном файле конфигурации.
- `config_create`: Сообщение о создании конфигурации.
- `config_created`: Подтверждение создания конфигурации.
- `please_set_config`: Просьба настроить конфигурацию.
- `exit`: Сообщение о выходе.
- `fail`: Сообщение об ошибке.
- `found_error_option`: Ошибка неверной опции.
- `missing_settings_directories`: Ошибка отсутствия settings или directories.
- `found_error_option_dir`: Ошибка неверной опции в директории.
- `missing_directory`: Ошибка отсутствующего каталога.
- `create_catalog`: Вопрос о создании каталога.
- `creating_catalog`: Сообщение о создании каталога.
- `found_language`: Сообщение о найденном языке.
