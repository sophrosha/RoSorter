# RoSorter - Обзор

## Описание
RoSorter - это инструмент для автоматической сортировки файлов по каталогам на основе их расширений и пользовательских правил. Программа поддерживает конфигурацию через YAML-файлы, имеет поддержку нескольких языков (русский и английский) и предназначена для работы на Windows (с планами поддержки Linux).

## Основные возможности
- **Гибкая конфигурация**: Настройка через YAML-файл с поддержкой множественных директорий и правил сортировки.
- **Правила именования**: Возможность создания кастомных названий для каталогов на основе расширений файлов.
- **Игнорирование файлов**: Список файлов, которые не будут сортироваться.
- **Многоязычность**: Поддержка русского и английского языков.
- **Логирование**: Опциональное ведение логов операций.
- **Тихий режим**: Работа без вывода уведомлений.

## Архитектура
Проект состоит из следующих модулей:
- `main.py`: Точка входа в приложение.
- `src/config.py`: Обработка конфигурационных файлов.
- `src/sorter.py`: Логика сортировки файлов.
- `src/languages.py`: Локализация сообщений.

## Требования
- Python 3.x
- Библиотеки: PyYAML, pystray, tkinter (для GUI)

## Статус проекта
Проект находится в разработке. Реализован базовый функционал сортировки для Windows. Планы включают:
- Графический интерфейс на Tkinter
- Демон для автоматической сортировки
- Поддержка Linux
- CLI интерфейс
- Сборка в исполняемый файл
