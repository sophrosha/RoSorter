# Использование RoSorter

## Запуск
После установки и настройки конфигурации запустите программу:
```bash
python main.py
```

Программа проверит конфигурацию, валидирует директории и выполнит сортировку файлов согласно правилам.

## Конфигурация
Основная конфигурация находится в YAML-файле. Пример конфигурации см. в `src/example.yaml`.

### Пример конфигурации
```yaml
settings:
  logs: false
  daemon: false
  timeout: "60m"
  gui: false
  silent: true
  language: "ru-RU"

directories:
  downloads:
    path: 'C:\Users\user\Downloads'
    files: ["png", "jpg", "pdf", "exe", "zip"]
    names:
      - "pdf": "Documents"
      - "exe": "Programs"
      - "*": "*"
    ignore:
      - "Telegram Desktop"
```

## Правила сортировки
- `files`: Список расширений для обработки. Используйте `["*"]` для всех файлов.
- `names`: Правила создания каталогов.
  - `"*" : "*"` - Создает каталоги по расширениям файлов.
  - `"pdf": "Docs"` - Перемещает PDF-файлы в каталог "Docs".
- `ignore`: Список файлов, которые будут пропущены.

## Логи
Если включено логирование (`logs: true`), файл `logs.txt` будет создан в каталоге программы.

## Режимы работы
- **Обычный режим**: Сортировка при запуске.
- **Демон** (планируется): Автоматическая сортировка в фоне.
- **GUI** (планируется): Графический интерфейс для настройки.

## Ошибки
Программа выводит сообщения об ошибках на выбранном языке. Основные ошибки:
- Отсутствие конфигурационного файла
- Неверный формат конфигурации
- Отсутствие указанных директорий
